## 📖 Visão Geral

O **IzziOrder** é uma aplicação web desenvolvida em **Next.js** e **Tailwind CSS**, estruturada seguindo os princípios da **Arquitetura Atomic Design**.  
Seu objetivo é **otimizar o processo de gerenciamento de pedidos e organização de fluxos de vendas**, garantindo:

- 🔹 **Escalabilidade**: estrutura modular que facilita evolução.  
- 🔹 **Reutilização**: componentes consistentes e reaproveitáveis.  
- 🔹 **Manutenção simplificada**: código limpo e organizado.  
- 🔹 **Performance**: experiência rápida e responsiva para o usuário.  

---

## 🏗️ Arquitetura do Projeto

O projeto segue os conceitos de **Atomic Design**, dividido em cinco camadas principais:

- **Atoms** → Elementos básicos (botões, inputs, labels).  
- **Molecules** → Combinações simples de átomos.  
- **Organisms** → Componentes mais complexos compostos por moléculas e átomos.  
- **Templates** → Estruturas que definem layouts reutilizáveis.  
- **Pages** → Páginas finais com conteúdo real.  

Essa abordagem melhora a **padronização**, **testabilidade** e **escalabilidade** da aplicação.

---

## 🚀 Tecnologias e Ferramentas

- **Next.js (App Router)** → Estrutura e renderização server-side.  
- **React** → Criação de componentes reativos e declarativos.  
- **TypeScript** → Tipagem estática para maior confiabilidade.  
- **Tailwind CSS** → Estilização rápida e responsiva.  
- **ESLint + Prettier** → Padrões de código e consistência.  
- **GitHub Actions (CI/CD)** → Integração e entrega contínua *(opcional se você já usa)*.
